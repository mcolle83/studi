Comment créer un compte ?

• Cliquez sur "Inscription", dans l'en-tête
• Renseignez vos informations personnelles : nom, prénom, email, mot de passe. Entrez le même mot de passe dans le champ "Confirmer mot de passe"
• Validez votre inscription en cliquant sur "Confirmer l'inscription"
• Vous recevrez un email d'activation de compte, contenant le lien vous permettant d'activer votre compte, ainsi qu'un QR Code vous permettant d'obtenir votre code OTP sur l'application Google Authenticator.

Comment se connecter ?

• Cliquez sur "Connexion", dans l'en-tête
• Renseignez vos informations personnelles : email, mot de passe, code OTP (code apparaisant sur l'application Google Authenticator)
• Cliquez sur "Se connecter"

Comment acheter un billet ?

• Cliquez sur "Liste des offres", dans l'en-tête
• Choisissez l'évènement qui vous intéresse
• Cliquez sur l'offre qui vous intéresse : unitaire, duo ou familial
• Votre sélection est mise dans le panier
• Cliquez sur l'image du panier, dans l'en-tête
• Vérifiez votre panier
• Procédez au paiement sécurisé en cliquant sur "Payer"
• Vous recevez un email de confirmation avec le e-ticket

Comment accéder à un évènement ?

• Présentez votre billet (QR Code) imprimé ou sur mobile
• Un scan du QR Code validera votre entrée

Besoin d'aide ?

• Contactez le service client en adressant un email à l'email fourni sur "Contact", dans le pied de page
